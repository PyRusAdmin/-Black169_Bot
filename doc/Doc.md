# The Black 169 — Telegram Bot

Телеграм-бот для кальянной The Black 169. Позволяет гостям регистрироваться, отслеживать бонусы, участвовать в акциях и
получать подарки. Интегрирован с системой лояльности QuickResto через API.

## Возможности

### Для пользователей:

- Регистрация пользователей по номеру телефона
- Приветственные бонусы (1000 ₽)
- Автоматическое создание клиентов в QuickResto
- Интеграция с QuickResto API (поиск, создание клиентов, получение бонусов)
- Главное меню с inline-кнопками
- Просмотр бонусов, акций, мероприятий
- Получение подарков (промокоды)
- 🎡 Колесо подарков (ежедневный розыгрыш, шанс 5%)
- Акция «Вернуться сегодня»
- Отслеживание сгорания бонусов (7, 3, 1 день)
- Поздравления с днём рождения (1500 бонусов)
- FSM для управления состояниями
- 🔐 Согласие на обработку персональных данных (ФЗ-152)

### Для администраторов:

- Админ-панель для управления
- 📊 Статистика пользователей
- 📨 Рассылка сообщений (текст, фото, видео)
- 👥 Выгрузка пользователей в Excel
- ✅ Выгрузка зарегистрированных пользователей в Excel
- 🏆 Выгрузка победителей «Колеса подарков» в Excel
- 🗑️ Удаление пользователей из QuickResto и базы данных бота
- 🎁 Управление промокодами (создание, просмотр, удаление)
- Поддержка нескольких администраторов

## Админ-панель

### Статистика пользователей

- Количество пользователей, запустивших бота (`start_persons`)
- Количество привязавших номер телефона (`registered_persons`)
- Статистика по рассылкам (всего, текстовых, с фото, с видео)
- Количество заблокировавших бота

### Рассылка сообщений

1. Выбор типа сообщения (текст / фото / видео)
2. Отправка контента боту
3. Подтверждение рассылки
4. Отправка всем пользователям из `start_persons`
5. Статистика: отправлено / заблокировано / не доставлено
6. Логирование в `marketing_messages`

### Выгрузка данных

- **Пользователи** — Excel-файл со всеми пользователями (ID, имя, фамилия, username, дата)
- **Зарегистрированные** — Excel-файл с зарегистрированными пользователями (все данные из QuickResto)
- **Победители** — Excel-файл с победителями «Колеса подарков» (ID, приз, дата)

### Удаление пользователей

- Удаление из QuickResto по ID
- Удаление из базы данных бота (обе таблицы: `registered_persons`, `start_persons`)

## Согласие на обработку персональных данных (ФЗ-152)

Бот соответствует требованиям Федерального закона № 152-ФЗ «О персональных данных».

### Логика работы

1. **Первый запуск** (`/start`):
   - Пользователь видит сообщение о обработке персональных данных
   - Необходимо нажать «✅ Даю согласие на обработку персональных данных»
   - Согласие фиксируется в таблице `consents` с датой и временем

2. **Повторный запуск**:
   - Если согласие уже дано — показывается главное меню
   - Если пользователь не зарегистрирован — предлагается отправить номер телефона

3. **Отказ от согласия**:
   - При нажатии «❌ Не даю согласие» — сообщение о невозможности использования бота
   - Предложение нажать `/start` для повторной попытки

### Какие данные обрабатываются

- Telegram ID
- Имя и фамилия
- Номер телефона
- Данные о бонусах и визитах (из QuickResto)

### Хранение данных

- Таблица `consents` — дата получения согласия
- Таблица `start_persons` — пользователи, запустившие бота
- Таблица `registered_persons` — зарегистрированные пользователи
- Интеграция с QuickResto — данные о клиентах в системе лояльности
